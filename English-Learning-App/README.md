# English-Learning-App
A web application for learning English — built with Angular (frontend) and Django (backend). Users can add vocabulary, take quizzes, and track their learning progress.
## Group Members
- Mustafayev Nurassyl
- Mustapa Akhmet
- Mustafa Orkenbek
## Features
- User authentication (login/logout)
- Add, view, and delete vocabulary
- Quiz system
- Track known and unknown words
## Technologies
- Frontend: Angular
- Backend: Django + Django REST Framework
## Pages
- Login
- Home
- Vocabulary
- Quiz
- Profile
## How To Run
### Frontend
Terminal
- cd frontend
- npm install
- ng serve
### Backend
Terminal
- cd backend
- pip install -r requirements.txt
- python manage.py runserver
